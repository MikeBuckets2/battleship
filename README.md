# Battleship
A simple Battleship game built for The Odin Project JavaScript curriculum.

## Features
 Play against the computer
 Random ship placement button
 Click the enemy board to attack
 Game ends when all ships are sunk

## Built With
 HTML, CSS, JavaScript
 Webpack
 Jest (for unit tests)

## Getting Started
1. Install dependencies:
   npm install

2. Run the dev server:
   npm run dev

3. Build for production:
   npm run build

## Running Tests
npm test

## Notes
This project focuses on game logic and unit testing. DOM testing is intentionally not included.
